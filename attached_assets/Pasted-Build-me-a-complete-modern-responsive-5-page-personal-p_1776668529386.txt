=== DESIGN REQUIREMENTS ===

1. COLOR SCHEME:
   - Background: Dark navy (#0a0a1a) or charcoal (#1a1a2e)
   - Primary accent: Electric blue (#00D4FF) or cyan
   - Secondary accent: Purple (#7B2FFE) for gradients
   - Text: White (#ffffff) and light gray (#b0b0b0)
   - Card backgrounds: Slightly lighter dark (#16213e)

2. TYPOGRAPHY:
   - Use Google Fonts: 'Inter' for body, 'Poppins' for headings
   - Clean, modern, readable

3. ANIMATIONS:
   - Smooth fade-in on scroll for sections
   - Card hover: lift up with box-shadow glow
   - Button hover: color shift with smooth transition
   - Navigation: smooth underline animation on hover
   - Page load: subtle fade-in animation

4. RESPONSIVE:
   - Mobile-first design
   - Hamburger menu on mobile
   - Cards stack to single column on mobile
   - All text readable on all screen sizes

5. GENERAL:
   - All external links open in new tab (target="_blank")
   - Active page highlighted in navigation
   - Consistent header/footer across all pages
   - Fast loading, minimal dependencies
   - SEO meta tags on each page
   - Favicon (use a simple emoji or generate one)
